# Raw Materials API List

This file lists all raw materials APIs currently implemented.

All endpoints below are protected (`JwtAuthGuard`) and use vendor context from the logged-in user.

## 1) Raw Materials Hazardous Products

- `POST /raw-materials-hazardous-products` - Create hazardous products record (per URN)
- `GET /raw-materials-hazardous-products/:urn_no` - List hazardous products records by URN
